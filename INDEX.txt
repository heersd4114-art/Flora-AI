    3. Use Local ResNet-50 (75-85% accurate)
    4. Return results either way (always works!)

  WiFi Mode Setup:
    1. Start Apache (port 80)
    2. Start MySQL (port 3306)
    3. Start Flora AI (port 5001)
    4. Phone connects via WiFi (no USB needed)

🔄 API ENDPOINTS (For Reference):

⚠️ KNOWN LIMITATIONS:

  1. Gemini Free Tier:
     • 20 requests/day limit
     • Resets at midnight UTC
     • Upgrade to paid for unlimited

  2. Local ResNet-50:
     • ~75% accuracy (vs 95% for Gemini)
     • Identifies plant category, not specific disease
     • Trade-off: Accuracy vs Availability

  3. WiFi Requirement:
     • Both devices must be on same network
     • Works on: Home WiFi, Office WiFi, Hotspot
     • Doesn't work: Cellular only, different networks
