# Technology Stack

## Frontend (Client-Side)
- **Languages**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Custom CSS (Vanilla) per user preference for visual excellence.
- **Frameworks**: None (Vanilla JS for maximum control), unless complex state management requires it later.

## Backend (Server-Side)
- **Language**: PHP 8.x
- **Web Server**: Apache (via XAMPP)

## Database
- **Engine**: MySQL / MariaDB
- **Management Tool**: phpMyAdmin

## Mobile Application (Future Phase)
- **Framework**: Flutter (Dart) or React Native (JavaScript/TypeScript)
- **Target**: Android (primary) & iOS

## AI/Machine Learning (Future Phase)
- **Model**: Python (Flask/FastAPI wrapper) or JavaScript-based (TensorFlow.js) depending on integration complexity.
- **Task**: Plant Disease Detection via Image Recognition.

## Tools & DevOps
- **Version Control**: Git
- **IDE**: VS Code
- **Local Env**: XAMPP
