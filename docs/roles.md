# Roles and Responsibilities

## User Roles (System)
1.  **Admin**: Full access. Manages users, plants, categories, delivery partners, and views system logs.
2.  **Customer**: Browses plants, adds to cart, places orders, tracks delivery, and manages profile.
3.  **Delivery Partner**: Views assigned shipments, updates delivery status (Picked Up, In Transit, Delivered).

## Team Roles (Development)
1.  **Solution Architect / Lead**:
    - Project Planning, DB Schema Design, Technology decisions.
2.  **Frontend Developer**:
    - HTML/CSS/JS implementation.
    - Responsive Design.
    - User Experience (UX) polish.
3.  **Backend Developer**:
    - PHP API development.
    - Database connectivity.
    - Logic implementation (Cart, Checkout, Auth).
4.  **Database Administrator (DBA)**:
    - SQL Optimization.
    - Backup & Recovery.
    - Data Integrity (Foreign Keys).
5.  **Mobile Developer**:
    - Android App development.
    - Integration with Backend APIs.
6.  **AI Specialist**:
    - Model selection/training for Plant Disease Detection.
    - Integration of model into the web/mobile app.
