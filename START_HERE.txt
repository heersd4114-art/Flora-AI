🆘 IF IT DOESN'T WORK:

❌ Problem: "Connection refused" on phone
   ✅ Solution:
     1. Run SETUP_WIFI_MODE.bat again
     2. Wait 60 seconds
     3. Restart app on phone
     4. Try again

❌ Problem: "Service unreachable"
   ✅ Solution:
     1. Both devices on SAME WiFi? Check!
     2. Run c:\xampp\htdocs\plant_app\VERIFY_SYSTEM.bat
     3. All services showing [OK]? (Apache, MySQL, AI)
     4. If not, run SETUP_WIFI_MODE.bat again

❌ Problem: "Not a plant" error
   ✅ Solution:
     1. Take clearer photo (full leaf visible)
     2. Better lighting (not too dark)
     3. Zoom in on the leaf
     4. Try scanning again

❌ Problem: Identification is inaccurate
   ✅ Reason: Using local AI (Gemini API quota exhausted)
   ✅ It's still working! Just less accurate
   ✅ Try multiple photos and compare results
